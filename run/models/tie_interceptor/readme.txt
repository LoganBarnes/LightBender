Tie Interceptor Model.
Created by Kehlan 2005.
Modeled in 3d max 7.
265,000 polygons triangulated.

Hope someone can find some use for this model. I think it might be a bit highpoly for
any games though. If your importing the model as an obj file, you may need to flip some of the normals around.
